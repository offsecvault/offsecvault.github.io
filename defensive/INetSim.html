 INetSim settings for malware analysis


1. edit the conf file
# sudo vi /etc/inetsim/inetsim.conf
2. uncomment "start_service dns" to enable the DNS usage
by default you will see [http,https,smtp.smtps,pop3,pop3s,ftp,ftps] - all are needed.
3. uncomment "service_bind_address", set the IP to: 0.0.0.0
4. uncomment "dns_default_ip", set the IP to: 10.10.10.3 [REMnux IP]
