 yarGen Tool 


yarGen, a very common YARA rule generator.
Project created by Neo23x0.

YARA Rule: meta, strings & condition

strings types: [$s = highly specific strings | $x = specific strings | $z = ordinary]

--> update the tool | it will regenerate the db needed for the correct usage.
# python3 yarGen.py --update

--> generate rules | it will generate a .yar file with the rules
# python3 tarGen.py -m /path/malware/sample

--> how to check the score of the rule ?
# python3 yarGen.py -m /path/to/.yar file --score

Flags:

-p = prefix for rule description
-a = author name
-r = reference URL or a text file
-m = path malware

# python3 yarGen.py -m /home/sample -p YARA -a b0ydC -r https://offsecvault.github.io --score
