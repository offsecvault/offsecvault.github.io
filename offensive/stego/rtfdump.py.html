rtfdump.py script

> get version
$ sudo python3 rtfdump.py --version
rtfdump.py 0.0.10

> get file overview
sudo python3 rtfdump.py [file].rtf
$ sudo python3 rtfdump.py odin.rtf

> select a level
sudo python3 rtfdump.py -s [number] [file].rtf 
$ sudo python3 rtfdump.py -s 1 odin.rtf

-s = hex result / hexshift

> decode hexadecimal
sudo python3 rtfdump.py -s [number] -H [file].rtf 
$ sudo python3 rtfdump.py -s 1 -H odin.rtf

more flags:

-d, --dump            perform dump
-x, --hexdump         perform hex dump
-a, --asciidump       perform ascii dump

download the python script: rtfdump.py script
