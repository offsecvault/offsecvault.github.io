 wpScan Usage 


::: normal scan

$ wpscan --url 

::: user enumeration

# wpscan --url https://example/ -e u

::: bruteforce

# wpscan --url https://example/ --passwords wordlist.txt --usernames b0ydC

-v = verbose
-o = output to a file
-u = users (under enumerate options)
--enumerate = feature to enumerate
--passwords or -P = password to use 
--usernames or -U = users to use
